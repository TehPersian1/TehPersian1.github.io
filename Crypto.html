Blockchain Technologies

CPSC 459
In this class, I researched various digital assets as a medium of exchange. I gained an understanding of decentralized and distributed ledgers that record verifiable transactions, as well as smart contracts, Etherium, Bitcoin, Mining, CryptoCurrency Ecosystems, Blockchain mechanics, and applications of the Blockchain.

Scrooge Coin

Our first project which introduced unspent transaction outputs in a pool. The pool is then verified to ensure that all outputs claimed by a transaction are in the output pool, the signatures are valid, there are no duplicate transactions, all outputs are non-negative, and the input value is >= output values.

  • Learn the Basics!

Consensus from Trust

The second project where we developed a distributed consensus algorithm given relationships between nodes. Nodes are either compliant or malicious, and we must design a compliant node that withstands an encounter of up to ~45% malicious nodes that do not cooperate.

  • Check it out!

The Block Chain

In this final project, we implemented a node that is part of a blockchain's distributed consensus protocol. We must receive incoming transactions, send outgoing transactions, and maintain an updated block chain. Processing happens one block at a time.
