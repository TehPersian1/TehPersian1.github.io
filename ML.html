Machine Learning

CPSC 481
In this class, I focused on working with real world data to both analyze and implement machine learning algorithms that focused on both statistical analysis and data analysis which granted the abilty to draw accurate conclusions about our data. We covered many techniques from both fields of supervised and unsupervised learning including: regression, classification, dimensionality reduction, clustering, and support vector machines. The class also familiarized me with many data science libraries, such as pandas, numpy, sklearn, matplotlib, and more!

Linear Regression

Our first project in machine learning which familiarized me with the basics of the libraries we will be utilizing, how to properly import and manipulate your data, as well as the basic techniques and algorithms involved in linear regression.

Dimensionality Reduction

In this project we were given a large dataset with over 20 differnt columns of various data types (categorical vs numeric) and were expected to use dimensionality reduction in order to examine only the important parts of data that are crucial to draw conclusions with.

Classification

My final project in my machine learning class, where we were given four uncategorized datasets with no labels, and were expected to use clustering techniques and support vector machines in order to utilize binomial classification to split our data into two groups. We examined the different algortihms along with different inputs. We were able to analyze and draw conclusions about which classification techniques were best for the different inputs.
