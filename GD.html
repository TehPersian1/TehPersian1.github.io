Game Design

CPSC 386
In this class I utilized Unreal Engine, Pygame, Magica Voxel, and Audacity to explore my creative skills in game design. The experience was complimented with research in the process of game design, development, and production.

Simple Physics Game

A simple physics and collision detection game in Unreal Engine. The project introduced how to create games with Blueprints and the basics of interacting with an Unreal Engine Scene structurally and programatically such as Procedural Generation.

  • Check it out!

Alien Invasion

My first project in Python. It introduced the basics of game development and game flow logic in python. The project introduced basics of game principles such as player movement, shooting, enemies, respawning, scores, and more by using the Pygame library.

  • Blast Aliens!

Crossy Road in Unreal Engine

A project where I created my assets in MagicaVoxel, Sounds in Audacity, and combined my prior knowledge of Unreal Engine physics, collision detection, particles, and procedural generation through blueprints to create my version of the endless arcade game crossy road.

  • Cross The Road!

Tower Defense in Unreal Engine

A semester-long group project utilizing the scrum method to cooperate and keep in touch with my team to realise our goal to design a tower defense game in Unreal Engine. The game utilizes AI targeting and pathing, custom map building using assets and level designers, wave design and logic flow, and UI design, all created through blueprints. It reinforced the importance of version control in a group environment.

  • Defend your Base!

PacMan Clone

My second Python project using Pygame. The project improved on the basics of game design described in alien invasion, compounded with the addition of animation, sound effects, collision testing, and the very basics of AI. It familiarized me with the process of making games by writing lines of code, and the logic behind doing so.

  • Escape the Ghosts!
